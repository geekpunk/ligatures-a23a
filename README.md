# A23a by Ligatures

The first LP named after the iceberg A23a. 12 tracks of punk, DCHC, hardcore, and noise core from Washington, DC.

Released September 1, 2024 on [Scene Police Records](https://scenepolicerecords.com) (SCP #42).

[Listen on Bandcamp](https://ligatures.bandcamp.com/album/a23a)
